<h3>Функционал</h3>
<p>Псевдо-импорт криптокошелька по сид фразе</p>
<p>Генерация сид фразы (рандомная выборка 12 слов из специального словаря), и последующая проверка по 3 словам</p>

<p>IDE: <b>Android Studio</b></p>
